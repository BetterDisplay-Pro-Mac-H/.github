## 📖 Overview

<a href="#betterDisplayPro">BetterDisplay Pro</a> gives Mac users deep control over display configuration that macOS System Settings does not expose — unlocking HiDPI resolution modes on external monitors, raising MacBook Pro brightness beyond the standard SDR limit, creating virtual displays for headless setups, and providing per-display configuration depth that power users and professionals need.

The <a href="#betterDisplayPro">HiDPI resolution unlocking</a> is BetterDisplay Pro's most impactful feature for external monitor users — macOS's System Settings only offers a limited set of scaled resolution options for any given display, and for many third-party monitors those options exclude HiDPI modes even when the display hardware is fully capable of rendering them sharply. BetterDisplay Pro accesses the display hardware's actual capabilities and unlocks resolution modes that System Settings omits, enabling sharp HiDPI scaled rendering on external monitors that macOS treats as standard definition by default. The visual difference between a non-HiDPI and a HiDPI scaled resolution on a capable 4K or 5K external monitor is significant — text and interface elements shift from acceptable-but-slightly-soft to genuinely sharp. The <a href="#betterDisplayPro">XDR brightness unlock</a> on compatible MacBook Pro models raises the built-in display brightness above the standard SDR maximum to the extended high-brightness level that Apple normally reserves for HDR content playback — making the display dramatically more legible in bright ambient lighting, direct sunlight, and outdoor use contexts where standard brightness is insufficient.

The <a href="#betterDisplayPro">virtual dummy display creation</a> generates software displays that macOS recognizes as real connected monitors — essential for Mac mini and Mac Pro setups accessed remotely without a physical display connected, where macOS would otherwise collapse all windows and spaces when no display is detected. Virtual displays maintain proper window arrangement and space configuration so remote access finds a properly organized desktop rather than all windows collapsed. The <a href="#betterDisplayPro">DDC hardware monitor control</a> adjusts physical monitor brightness, contrast, and other DDC-compatible settings directly from the Mac menu bar — using the Display Data Channel protocol to send hardware control commands to the monitor without touching the physical OSD buttons.

<a href="#betterDisplayPro">Per-display color temperature and night mode</a> applies independent color warmth settings to each connected display on its own schedule — different color temperature values and timing for each monitor rather than the system-wide Night Shift that affects all displays identically. <a href="#betterDisplayPro">Display override profiles</a> apply custom EDID metadata and identification overrides to correct misidentified monitors, enable features restricted to specific approved display models, and fix compatibility issues with displays that macOS handles incorrectly.

<a href="#betterDisplayPro">Picture-in-Picture display mode</a> creates a floating overlay window showing any connected display's content — monitoring a secondary display from the primary screen without switching focus or physically looking at the second screen. <a href="#betterDisplayPro">Display arrangement memory</a> saves and automatically restores display configurations when monitors connect and disconnect — the layout restores correctly each time a specific monitor set connects rather than requiring manual repositioning.

## 💬 FAQ

<details>
<summary>What is HiDPI and why does it matter on external monitors?</summary>

HiDPI renders the Mac interface at double pixel density for sharp text and graphics — the same technology that makes Retina displays look sharp. macOS restricts HiDPI modes to approved displays; BetterDisplay Pro unlocks them on capable external monitors that macOS excludes.

</details>

<details>
<summary>Does the XDR brightness unlock work on all MacBook Pro models?</summary>

XDR brightness unlock is available on MacBook Pro models with Liquid Retina XDR displays. Check current BetterDisplay Pro documentation for specific model compatibility.

</details>

<details>
<summary>What is a virtual dummy display?</summary>

A virtual dummy display is a software-created display that macOS treats as a real connected monitor — maintaining window layouts and spaces even when no physical display is connected to the Mac.

</details>

<details>
<summary>Does BetterDisplay Pro control monitor brightness through software?</summary>

Yes. DDC hardware control sends brightness and other setting commands to compatible monitors directly from the Mac menu bar without using the monitor's physical OSD buttons.

</details>

<details>
<summary>Can BetterDisplay Pro set different color temperatures per display?</summary>

Yes. Independent color temperature values and schedules apply to each display rather than the system-wide Night Shift that affects all displays identically.

</details>

<details>
<summary>What is a display override profile?</summary>

Display override profiles apply custom EDID identification data to correct monitors that macOS misidentifies, enabling features that macOS restricts to specific approved display models.

</details>

<details>
<summary>Does BetterDisplay Pro remember display arrangements?</summary>

Yes. Display arrangement memory saves layouts and automatically restores them when specific monitor configurations connect.

</details>

<details>
<summary>Is BetterDisplay Pro available through Setapp?</summary>

Yes. BetterDisplay Pro is available through the Setapp subscription service.

</details>

<details>
<summary>Does BetterDisplay Pro run on Apple Silicon?</summary>

Yes. Current versions support Apple Silicon Macs natively.

</details>
